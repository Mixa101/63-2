# 📝 Контрольная работа

---

## 📋 Описание проекта

Необходимо реализовать функционал блога с системой авторизации пользователей. Пользователи должны иметь возможность регистрироваться, создавать посты и оставлять комментарии.

---

## ✅ Требования к функционалу

### 1. Модели данных
- Реализовать модель `Post` (Заголовок, Текст, Изображение, Дата создания, Автор) (то что уже есть у вас).
- Реализовать модель `Comment` (Текст, Автор, Дата создания, Связь с Постом).
- Настроить связи между моделями (Пост ↔ Пользователь, Комментарий ↔ Пост).
- Зарегистрировать модели в админ-панели.

### 2. Авторизация и регистрация
- Реализовать страницу регистрации нового пользователя ().
- Реализовать страницу входа (Логин).
- Реализовать выход из аккаунта (Логаут).
- Валидация паролей (совпадение паролей при регистрации).

### 3. Управление постами
- Страница списка всех опубликованных постов.
- Страница детального просмотра одного поста.
- Страница создания нового поста (форма с заголовком, текстом и загрузкой изображения).
- **Ограничение:** Создание постов доступно **только авторизованным** пользователям.
- При создании поста автор должен сохраняться автоматически (текущий пользователь).

### 4. Комментарии
- На странице детального просмотра поста должен быть список комментариев.
- Форма добавления комментария доступна на странице поста.
- Комментарий должен содержать имя автора и текст.

### 5. Интерфейсы

- Максимально просто чтобы можно было видеть что есть пользователь, комментарии и посты

## ⚙️ Технические требования

- Проект должен запускаться без ошибок после команды `python manage.py runserver`.
- Все миграции должны быть применены.
- Файлы медиа (изображения) должны загружаться и отображаться.
- В репозитории **не должно быть** файлов: `db.sqlite3`, `venv/`, `media/`, `__pycache__/`, `.pyc`.
- Наличие файла `.gitignore` обязательно.
- Наличие файла `requirements.txt` обязательно.
